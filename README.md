# Data Validation & Reconciliation Analysis (Excel)

## 📌 Overview
This project simulates a real-world business scenario where raw transactional data contains inconsistencies, errors, and missing values. The objective is to clean, validate, and reconcile the data to ensure accuracy and reliability.

## 🎯 Objectives
- Identify and fix data quality issues
- Perform validation checks on transactions
- Reconcile mismatched records
- Build a structured reporting system

## 🛠 Tools Used
- Microsoft Excel
- Functions: XLOOKUP, IF, TRIM
- Pivot Tables
- Data Cleaning Techniques

## 📂 File Structure
- Raw_Data → Contains inconsistent data (duplicates, missing values)
- Cleaned_Data → Processed and corrected dataset
- Validation_Checks → Error detection logic
- Reconciliation_Summary → Final validated results
- Dashboard → Visual summary of issues and fixes

## 🔍 Key Insights
- Identified duplicate transaction records
- Detected mismatched total values
- Improved overall data accuracy
- Built reusable validation framework

## 🚀 Outcome
This project demonstrates strong data cleaning, validation, and reconciliation skills required for data analyst and business analyst roles.
